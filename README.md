# The Effect of Source Prestige on Perceived Credibility of Health Information
This repository contains the published HTML version of my PSY120L final reproducible report. 
The intention of this study is to examine whether the prestige of a source influences perceived credibility of health information and whether this effect depends on participants’ need for cognition and/or trust in science.
This report was created using Quarto and R.
The published report file is index.html.
The dataset is not included in this repository.
